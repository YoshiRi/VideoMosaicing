概要：動画の各フレームから画像の変位を抽出して動作の推定を行うこと。
イメージモザイキングを用いてそれを評価する。


1:ExtractData：動画から画像を抽出して，そこからフレーム間の変形量を取り出す。
2:Optimization：取り出した変形量と評価値のデータから尤もらしいカメラの動きを推定。
3:Mosaicing：イメージモザイキングを行う。
4:Evaluation:モザイキングの結果からカメラの動きの推定精度を評価する。



1.ではPOCを用いている。一対応当たり1秒くらい。
4.のEvaluationの際に評価用のGround Truthが必要。
今回はFrame1の広域画像を使っている。行列の対応は手打ち。
